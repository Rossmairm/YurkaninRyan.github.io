#Getting it running locally

* First, open up a terminal.

* Install Node.js, giving you Node Pacakage Manager.

* git clone this repo.

* Navigate into the directory and issue the following commands:

--1. npm install
--2. bower install
--3. gulp

* Open Index.html and view it!
